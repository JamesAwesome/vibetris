# Vibetris Logo Asset

This directory should contain the Vibetris logo image.

## Required Asset

**File:** `vibetris-logo.png`

**Specifications:**
- Format: PNG with 1-bit depth (black and white)
- Dimensions: Approximately 300x80 pixels (fits Playdate 400x240 screen)
- Style: Bubble letters - rounded, three-dimensional appearance
- Content: The word "VIBETRIS" in stylized bubble lettering
- Compatibility: Playdate SDK image format requirements

## Creating the Logo

The logo should be created using external image editing tools such as:
- Aseprite (recommended for pixel art)
- Photoshop
- GIMP
- Any tool that can export 1-bit PNG

## Fallback Behavior

If the logo asset is not present, the game will automatically fall back to text-based rendering of "VIBETRIS" on the start screen.

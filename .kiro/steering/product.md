# Product Overview

Vibetris is a classic block-stacking puzzle game (Tetris) built for the Playdate handheld console. The game features all 7 standard tetrominos with traditional gameplay mechanics adapted for Playdate's unique controls.

## Key Features

- Dual rotation input: Playdate crank or B button
- Traditional controls: D-pad for movement, A/Up for hard drop
- Progressive difficulty with level-based fall speed scaling
- Line clearing with visual feedback animation
- Score tracking and level progression
- Pause functionality
- Start screen with menu navigation

## Target Platform

Playdate handheld console - a unique gaming device with:
- 1-bit black and white display (400x240 pixels)
- Crank input mechanism
- D-pad and A/B buttons
- Limited resources requiring optimized code

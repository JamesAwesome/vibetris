# Tetris for Playdate

A classic Tetris game built for the Playdate console using the Playdate Lua SDK.

## Setup

1. Install the [Playdate SDK](https://play.date/dev/)
2. Set the `PLAYDATE_SDK_PATH` environment variable to your SDK location

## Building

```bash
pdc source Tetris.pdx
```

## Running

Open the generated `Tetris.pdx` file with the Playdate Simulator, or run:

```bash
open Tetris.pdx
```

## Development

The main game code is in `source/main.lua`. The Playdate SDK documentation is available at https://sdk.play.date/
